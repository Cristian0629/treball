Servidor d'aplicacions Apache Tomcat (c)


Inici | Descàrregues | Documentació

Apache Tomcat és un servidor web de codi obert i un contenidor de servlets desenvolupat per la Apache Software Foundation. S'utilitza principalment per a executar aplicacions web basades en Java. Aquest servidor és conegut per la seva lleugeresa i eficiència, i és àmpliament utilitzat per a desplegar aplicacions web dinàmiques i servlets Java.
